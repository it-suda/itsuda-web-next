# itsuda-web-next

> IT 이벤트 공유 서비스 "잇수다" (Web Frontend)

### Homepage

- https://itsuda-web-next.vercel.app

### Features

#### v1.0.0

- 메인 페이지
  - 배너
  - 이벤트 보기
    - 캘린더로 보기 (월간)
    - 카드형 목록으로 보기
    - 카테고리별 목록 필터링
- 상세 페이지
  - 이벤트 상세 정보

### Stack

- Next.js
- TypeScript
- styled-components

### Command

```bash
npm i
npm run dev # Open http://localhost:3000 with your browser to see the result.
```
